# Security Policy

This repository primarily distributes map boundary data (GeoJSON/TopoJSON).

## Reporting a vulnerability

If you believe you have found a security issue (for example, a maliciously crafted file or supply-chain concern), please:

- Avoid opening a public issue with exploit details.
- Contact the repository owner via GitHub with a description and steps to reproduce.

## Supported versions

Security fixes apply to the current contents of the default branch.
